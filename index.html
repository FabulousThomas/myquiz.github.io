MyQuiz Blog

Lack of access to adequate practice questions/tests with detailed solutions is a common problem among students.
MyQuiz - a mobile app and website - is set to bridge this gap by providing students access to a large database of quizzes, practice tests/examinations, past questions, and other learning resources.

MyQuiz Blog

The use of MyQuiz, however, is not limited to students but other learners or professional across other fields In addition, MyQuiz is a medium where people can share quizzes created by them with their learning peers or general public using a unique pin.

Are You Ready To Get Started?

MyQuiz has the following features:
1. Provides access to a large database of questions (academics, technology, programming e.t.c). The number of quizzes/questions increases as users create questions.
2. Users create and share questions, quizzes, practice tests or examinations.
3. Downloading of resources ( pdf / images / videos). Links or URLs to resources are also available.
4. View Results / Leadership-board
5. Users can rate/give feedback on a quiz(we use this to determine spam / fake quiz)
6. Support for true / false options 3 - 5 options
7. Users can share their results with friends on social media.
