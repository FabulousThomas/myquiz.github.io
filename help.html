Lack of access to adequate practice questions/tests with detailed solutions is a common problem among students. MyQuiz - a mobile app and website - is set to bridge this gap by providing students access to a large database of quizzes, practice tests/examinations, past questions, and other learning resources. The use of MyQuiz, however, is not limited to students but other learners or professional across other fields In addition, MyQuiz is a medium where people can share quizzes created by them with their learning peers or general public using a unique pin.

MyQuiz has the following features:

1. Provides access to a large database of questions (academics, technology, programming e.t.c). The number of quizzes/questions increases as users create questions.
2. Users create and share questions, quizzes, practice tests or examinations.
3. Downloading of resources ( pdf / images / videos). Links or URLs to resources are also available.
4. View Results / Leadership-board
5. Users can rate/give feedback on a quiz(we use this to determine spam / fake quiz)
6. Support for true / false options 3 - 5 options
7. Users can share their results with friends on social media.

There are 3 quiz modes:

Exam mode, Practice mode, and Quiz mode.
The characteristics of the different quiz modes are enumerated below.

Exam mode

1. Only the creator can view users result
2. No answers / explanation
3. Mini calculator available
4. Timer Support
Practice mode

1. Results and Leadership-board available for all users
2. Answers and explanations would be given after submitting
3. Timer support
4. Mini Calculator available
Quiz mode

1. Results and Leadership-board available for all users
2. Answers and explanation given immediately after each question
3. Timer for each question
Use case scenario

1. Jamb / Waec / Post Utme candidates can make use of it to practice for their exams
2. University students / Tutorial masters can make use of the different modes to help themselves
3. Digital skills trainees
